This is the API

domestic

URL

https://apiHost//v1.0/payments/v1.0/domestic

HTTP Method

POST

Response Type

AddPaymentResponse

Parameters

  • Body: body PaymentDomesticRequest(PaymentDomesticRequest)

    Data for domestic transfer

  • Header: Authorization String

    The value of the Authorization header should consist of 'type' + 'credentials', where for the approach using the 'type' token should be 'Bearer'.

  • Header: Accept-Encoding String

    Gzip, deflate

  • Header: Accept-Language String

    Prefered language of response

  • Header: Accept-Charset String

    UTF-8

  • Header: X-JWS-SIGNATURE String

    Detached JWS signature of the body of the payload
