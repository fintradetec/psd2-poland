This is the API

getConfirmationOfFunds

Confirming the availability on the payers account of the amount necessary to execute the payment transaction, as defined in Art. 65 PSD2.

URL

https://apiHost//v1.0/confirmation/v1.0/getConfirmationOfFunds

HTTP Method

POST

Response Type

ConfirmationOfFundsResponse

Parameters

  • Body: body ConfirmationOfFundsRequest(ConfirmationOfFundsRequest)

    Object with amount to check

  • Header: Accept-Encoding String

    Gzip, deflate

  • Header: Accept-Language String

    Prefered language of response

  • Header: Accept-Charset String

    UTF-8

  • Header: X-JWS-SIGNATURE String

    Detached JWS signature of the body of the payload
