This is the API

authorize

Requests OAuth2 authorization code

URL

https://apiHost//v1.0/auth/v1.0/authorize

HTTP Method

POST

Response Type

Parameters

  • Body: body AuthorizeRequest(AuthorizeRequest)

    Data for OAuth2 Authorization Code Request

  • Header: Accept-Encoding String

    Gzip, deflate

  • Header: Accept-Language String

    Prefered language of response

  • Header: Accept-Charset String

    UTF-8

  • Header: X-JWS-SIGNATURE String

    Detached JWS signature of the body of the payload

authorizeExt

Requests OAuth2 authorization code based One-time authorization code issued by External Authorization Tool. Authorization code will be delivered to TPP as callback request from ASPSP if PSU authentication is confirmed by EAT. Callback function must provide similar notification also in case of unsuccessful authentication or its abandonment.

URL

https://apiHost//v1.0/auth/v1.0/authorizeExt

HTTP Method

POST

Response Type

Parameters

  • Body: body EatCodeRequest(EatCodeRequest)

    Data for OAuth2 Authorization Code Request extended for EAT based authentication and callback response

  • Header: Accept-Encoding String

    Gzip, deflate

  • Header: Accept-Language String

    Prefered language of response

  • Header: Accept-Charset String

    UTF-8

  • Header: X-JWS-SIGNATURE String

    Detached JWS signature of the body of the payload
